## 🎨 How to Customize (2 Minutes)

### Step 1: Go to Admin Customization

1. Login: http://localhost:8000/admin/
2. Left sidebar এ দেখবে: **"Admin Interface"**
3. Click করো: **"Themes"**
4. Click করো: **"Django"** (default theme)

### Step 2: Customize Your Theme

এখন একটা page open হবে যেখানে সব কিছু customize করতে পারবে:

#### 🎨 Colors Section

**Title** - Admin panel এর title bar color:
- Click করো color picker এ
- আপনার পছন্দের color choose করো
- Example colors:
  - Blue: `#0066FF`
  - Green: `#00CC66`
  - Purple: `#9933FF`
  - Orange: `#FF6600`
  - Red: `#FF0033`

**Header** - Top header background:
- Same way, color picker দিয়ে change করো

**Links** - Link colors:
- Normal link color
- Hover link color
- Selected link color

**Button** - Button colors:
- Save button color
- Hover effects

**Module** - Dashboard module colors

#### 🖼️ Logo Section

**Logo:**
- আপনার logo upload করো (PNG/SVG)
- Max height set করো (default 35px)
- Width automatically adjust হবে

**Favicon:**
- Browser tab এ show হবে
- .ico বা .png file upload করো

#### 🏷️ Environment Badge

**Environment name:**
- "Development" বা "Production" লিখো

**Environment color:**
- Development: Green/Blue
- Production: Red/Orange

**Visible:** 
- Checkbox check করো show করতে চাইলে

## 🚀 Quick Customization (30 Seconds)

1. Go to: http://localhost:8000/admin/admin_interface/theme/1/change/
2. Change these 3 colors:
   - **Title color**: `#1E3A8A`
   - **Header color**: `#3B82F6`
   - **Link color**: `#2563EB`
3. Click **"Save"**
4. Refresh your admin panel!

## 💡 Pro Tips

### 1. Color Picker Tips
- Click color field → Color picker opens
- Use hex codes directly: `#FF6600`
- Preview shows instantly

### 2. Logo Best Practices
- Use PNG with transparent background
- Recommended height: 30-40px
- SVG works best for scaling

### 3. Environment Badges
- **Development**: Green (#10B981)
- **Staging**: Yellow (#F59E0B)
- **Production**: Red (#EF4444)

### 4. Test Before Apply
- Save এর আগে preview দেখো
